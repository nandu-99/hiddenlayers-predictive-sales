# Datasets

This directory contains the datasets used for the predictive sales project.

## Final Datasets

To avoid bloating the repository size, the large dataset files are not tracked by Git. You can download the datasets directly from the following sources when needed and place them in the `raw/` directory:

1. **SaaS Sales Conversations:** [DeepMostInnovations/saas-sales-conversations](https://huggingface.co/datasets/DeepMostInnovations/saas-sales-conversations)
2. **CRM Sales Opportunities:** [Maven Analytics CRM Sales Opportunities](https://mavenanalytics.io/data-playground/crm-sales-opportunities)

## Directory Structure

- `raw/`: Place the original, unmodified datasets downloaded from the links above here.
- `processed/`: This folder will contain the cleaned and preprocessed datasets ready for modeling.
